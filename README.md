# Flutter Movies
一款基于豆瓣电影 Api 使用 Flutter 开发的移动端应用。
- Flutter: Channel stable, 1.22.3
- Dart: 2.10.3
- 运行环境：iOS 12.0 或以上（暂未适配 Android）

## 版本
