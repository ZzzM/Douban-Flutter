# Flutter Movies

A movies news App developed by Flutter（ ⚠️ **API is no longer available** ）


## Features
- [x] Switching theme dynamically
- [x] Internationalization (简体中文、English)
- [x] Video, photo download
- [x] Video play 


## Compatibility
- Requires **iOS 12.0** or later
